# Slack "Skintonator"

A PHP cli script to skintone the fuck out of 16x16 PNGs.

# Usage

    php Skintonator.php ./test-images/heart.png

# What it does

Turn any 16x16 png (_or any other png, until it crashes_) to a slack image made of `:skin-tone:`  emojis.
